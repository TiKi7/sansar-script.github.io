SetIsVisible Method

Sets the visibility state of this MeshComponent.

Syntax

[Sansar.Script.Interface]
public void SetIsVisible (Sansar.Script.SessionId sessionId, bool isVisible)

Parameters

sessionId
The sessionId of the user to change static mesh visibility for.
isVisible
The visibility state to set.

Remarks

It will only change static mesh visibility for a single user and will not persist across client restarts, server restarts, or leaving the scene (even for changing avatars).
